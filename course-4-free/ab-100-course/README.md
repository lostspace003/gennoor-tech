# Gennoor AB-100 — Course Bundle

Self-paced course bundle for the Microsoft AB-100 (Architecting Agentic AI Business Solutions) certification. Fully Gennoor-branded.

> **💡 Tip:** Press **F11** in your browser for full-screen viewing of any HTML file in this bundle.

---

## What's inside

### `chapters/` — 13 interactive HTML slide decks
Each chapter has a slide navigator centered in the topbar with prev/next arrows and a slide counter. All content rendered at 125% font scale for comfortable reading. Open in any modern browser. Run offline.

| File | Coverage |
|------|----------|

| `chapter-03-strategy.html` | Develop AI Strategy · CAF · agent types · solution rules |
| `chapter-04-roi.html` | Return on AI Investment · TCO · model router |
| `chapter-05-design.html` | Design AI Solutions · RAI · Copilot Studio · Foundry |

**Inside any chapter:** topbar shows brand on the left, the **slide navigator** (prev / counter / next) in the center, section label and Notes button on the right.

**Keyboard:** `T` notes panel · `P` progress bar · `←/→` advance step / navigate slides.

### `notes/` — 13 Word documents
Speaker notes per chapter. One `.docx` per chapter.
