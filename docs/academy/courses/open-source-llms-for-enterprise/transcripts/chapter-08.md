> [S4 ▸ reveal 1 · 4 trust trip-wires]

Four trust trip-wires. *Do not cross.*

*Trip-wire one — selling on cost savings alone.* Sovereignty and control are the real cases. Cost takes 18-24 months.

*Trip-wire two — fine-tuning when retrieval would solve it.* Cheap shortcut to demonstrated value, expensive shortcut to wrong solution.

*Trip-wire three — laptop benchmarks as production capacity.* Real concurrency-degraded TPS is 25-50% of benchmark numbers.

*Trip-wire four — no patch + upgrade discipline.* Self-hosted LLM with stale dependencies is a CVE waiting to be exploited.
