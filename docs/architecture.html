3 · Identity (two separate systems)

🛡️ Admin Auth — NextAuth v5
Microsoft Entra ID OAuth/OIDC → email allow-list (AZURE_AD_ALLOWED_USERS) → session cookie, middleware-guarded /admin/*
🎓 Learner Auth — custom JWT (jose)
Providers: Email OTP · Google OAuth · Microsoft → signed HS256 learner-session cookie · 30 days

5 · Key End-to-End Flows

AI Readiness Form→ /api/ai-readiness/generate→ OpenAI blueprint→ Render PDF → Blob→ Email report (ACE)→ Persist lead + track (Table)
Expert Call GET availability→ MS Graph free/busy→ POST create booking→ Graph appointment→ Confirmation email
Certificate Complete course→ PDFKit + QR→ Upload Blob (SAS)→ Save record (Table)→ Public /verify/[certId]
Academy Content Request slide/audio (Range)→ /api/content/[...path]→ Blob range stream (206)→ Save progress (local)→ /api/learner/progress/sync
