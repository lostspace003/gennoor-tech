Course Curriculum

5-Day Curriculum

Covers all 13 Claude courses with instructor-led depth. Schedule and session order are fully customizable based on your organization's needs and time zones.

01
Claude Foundations & AI Fluency
Covers: Claude 101 · AI Fluency: Framework & Foundations · AI Fluency for Educators · Students · Nonprofits · Teaching AI Fluency
Session A
Claude 101 & AI Fluency Foundations
  • Claude 101: capabilities, interface, prompt craft, attachments, projects
  • AI Fluency Framework: what AI can and cannot do, mental models
  • Effective human-AI collaboration patterns and workflows
  • Lab: Claude for business analysis, writing, research, and summarization
  • Lab: Build a personal AI workflow for your daily role
Session B
AI Fluency for Teams — Education, Nonprofits & Teaching
  • AI Fluency for Educators: curriculum design, assessment with AI
  • AI Fluency for Students: research, learning, and academic integrity
  • Teaching AI Fluency: designing AI literacy programs for organizations
  • AI Fluency for Nonprofits: grant writing, impact reports, donor comms
  • Workshop: Design an AI fluency rollout plan for your organization
02
Building with the Claude API — Part 1
Covers: Building with the Claude API (84 lectures, first half)
Session A
Claude API Fundamentals
  • Messages API: roles, system prompts, multi-turn conversations
  • Model selection: Opus, Sonnet, Haiku — capabilities and pricing
  • Streaming responses, token counting, and rate limit handling
  • Lab: Build a Claude chatbot with the Python SDK
  • Lab: Build a multi-turn conversation app with conversation memory
Session B
Advanced API Patterns
  • Tool use and function calling with Claude
  • Vision: analyzing images and documents with Claude
  • Structured output: JSON mode, XML parsing, schema enforcement
  • Lab: Build a tool-calling assistant that queries external APIs
  • Lab: Build a document analysis pipeline with vision + structured output

03
Claude API Part 2 + Claude Code
Covers: Building with the Claude API (second half) · Claude Code in Action · Introduction to Agent Skills
Session A
API: Prompt Engineering, Caching & Evaluation
  • Advanced prompt engineering: chain-of-thought, few-shot, system prompts
  • Prompt caching for cost and latency optimization
  • Evaluation and testing: building evals for AI applications
  • Content moderation and safety with Claude
  • Lab: Build an evaluation harness for a Claude-powered application
Session B
Claude Code in Action & Agent Skills
  • Claude Code: terminal-based agentic coding assistant
  • Project scaffolding, code generation, debugging, and refactoring
  • Introduction to Agent Skills: building custom agent capabilities
  • Lab: Use Claude Code to build a complete web application
  • Lab: Create custom agent skills for code review and testing
04
Model Context Protocol (MCP)
Covers: Introduction to MCP · MCP: Advanced Topics
Session A
Introduction to Model Context Protocol
  • What is MCP? Architecture, transports, and protocol design
  • MCP primitives: tools, resources, prompts, and sampling
  • Building MCP servers with Python and TypeScript
  • Lab: Build your first MCP server with tools and resources
  • Lab: Connect an MCP server to Claude Desktop
Session B
MCP: Advanced Topics
  • Advanced MCP patterns: SSE transport, authentication, security
  • Building MCP servers for databases, APIs, and file systems
  • MCP server discovery, configuration, and deployment
  • Lab: Build an enterprise MCP server connecting to a SQL database
  • Lab: Build a multi-tool MCP server with authentication
05
Claude on Cloud Platforms & Capstone
Covers: Claude with Amazon Bedrock · Claude with Google Cloud's Vertex AI
Session A
Claude with Amazon Bedrock & Google Vertex AI
  • Claude on Bedrock: Converse API, Knowledge Bases, Agents, guardrails
  • Claude on Vertex AI: model deployment, grounding, extensions
  • Cross-platform patterns: when to use direct API vs Bedrock vs Vertex
  • Lab: Deploy Claude on Bedrock with Knowledge Bases for RAG
  • Lab: Deploy Claude on Vertex AI with enterprise grounding
Session B
Capstone: End-to-End Claude Application
  • Team project: Build a complete Claude-powered application
  • Combine API + MCP + Claude Code + cloud deployment
  • Architecture review: cost optimization, security, monitoring
  • Team presentations and expert feedback
  • Completion of all course modules completed
Flexible & Customizable: Session order, timing, depth, and industry focus can be tailored to your organization's specific requirements. For developer-only audiences, AI Fluency modules can be replaced with deeper API and MCP labs.
