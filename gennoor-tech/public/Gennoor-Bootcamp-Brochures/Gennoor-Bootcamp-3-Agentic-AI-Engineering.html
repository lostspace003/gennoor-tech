Agentic AI Engineering

Course Overview

This 5-day technical bootcamp takes developers and architects from understanding AI agent fundamentals to building production-grade multi-agent systems using LangChain, LangGraph, CrewAI, Semantic Kernel, Microsoft Copilot Studio, and Model Context Protocol (MCP) servers. Participants build 6 real-world agent projects by the end of the program.

This is the most in-demand AI skill in 2026. Enterprises are moving beyond chatbots to autonomous AI agents that reason, plan, use tools, and orchestrate workflows. This bootcamp gives your team the hands-on ability to design, build, and deploy these systems on Azure, AWS, and open-source stacks. Designed and delivered by an AB-100 certified Agentic AI architect who has built production MCP servers.

Learning Outcomes

1
Build AI agents with LangChain & LangGraph — design ReAct agents, tool-calling agents, and stateful graph-based workflows
2
Orchestrate multi-agent systems with CrewAI — create teams of specialized agents that collaborate, delegate, and reason together
3
Build MCP servers from scratch — implement Model Context Protocol servers that connect AI agents to databases, APIs, and enterprise systems
4
Deploy agents on Microsoft Copilot Studio — build enterprise-grade Copilot agents with Dataverse, Power Automate, and Azure AI integration
5
Implement RAG pipelines for agents — ground agents in enterprise knowledge using Azure AI Search, FAISS, Chroma, and document intelligence
6
Design production agent architectures — handle memory, state, human-in-the-loop, error recovery, guardrails, and monitoring

Prerequisites

Python programming (intermediate), basic understanding of REST APIs and JSON, familiarity with at least one cloud platform (Azure/AWS/GCP). Prior ML/AI experience is helpful but not required.

Course Curriculum

5-Day Curriculum

Interactive format with live coding, hands-on labs, and project-based learning. Schedule and session order are fully customizable based on your organization's needs and time zones.

01
AI Agent Foundations — LangChain & Tool Calling
Lecture + Live Coding + Hands-On Labs
Session A
Understanding AI Agents
  • What are AI agents? From chatbots to autonomous systems
  • Agent architectures: ReAct, Plan-and-Execute, Reflexion
  • The agentic AI stack: LLMs, tools, memory, orchestration
  • Azure OpenAI vs Claude vs open-source models for agents
  • Lab: Set up development environment (Python, LangChain, API keys)
Session B
LangChain Agents & Tool Calling
  • LangChain core: chains, prompts, output parsers
  • Building tool-calling agents with function calling
  • Custom tools: web search, database queries, API calls
  • Lab: Build a research agent that searches, summarizes, and reports
  • Lab: Build a SQL agent that queries databases via natural language
02
LangGraph — Stateful Multi-Step Agent Workflows
Live Coding + Hands-On Labs + Architecture Workshop
Session A
LangGraph Fundamentals
  • Graph-based agent orchestration: nodes, edges, state
  • Conditional routing and branching in agent workflows
  • Persistence: checkpoints, memory, and state management
  • Lab: Build a multi-step document processing agent
  • Lab: Build a customer support agent with escalation logic
Session B
Human-in-the-Loop & Advanced Patterns
  • Human-in-the-loop: approval gates, feedback loops
  • Error handling, retries, and fallback strategies
  • Streaming responses and real-time agent updates
  • Lab: Build a loan approval agent with human review step
  • Lab: Build a content moderation pipeline with multi-stage review

03
CrewAI & Multi-Agent Collaboration
Live Coding + Hands-On Labs + Architecture Design
Session A
CrewAI Framework Deep Dive
  • CrewAI architecture: agents, tasks, crews, and processes
  • Defining agent roles, goals, and backstories
  • Sequential vs hierarchical crew execution
  • Lab: Build a financial research crew (analyst + writer + reviewer)
  • Lab: Build an automated hiring pipeline with specialized agents

Semantic Kernel & Microsoft Agent Framework
  • Semantic Kernel: plugins, planners, and memory
  • Microsoft Agent Framework and Azure AI Agent Service
  • AutoGen: multi-agent conversation patterns

MCP Servers & Enterprise Integration
Live Coding + Hands-On Labs + Production Patterns
Session A
Model Context Protocol (MCP) Servers
  • What is MCP? Architecture, tools, resources, and prompts
  • Building MCP servers with Python and FastAPI
  • Connecting MCP servers to databases (SQL, Cosmos DB)
  • Lab: Build an Azure SQL MCP server from scratch
  • Lab: Build a document retrieval MCP server with AI Search
Session B
Copilot Studio & Enterprise Deployment
  • Connecting MCP servers to Microsoft Copilot Studio
  • Building enterprise agents with Dataverse integration
  • Power Automate flows triggered by AI agents
  • Lab: Deploy MCP server to Azure and connect to Copilot Studio
  • Lab: Build insurance claims processing agent (end-to-end)

RAG, Production Architecture & Capstone
Capstone Project + Architecture Review + Presentations
Session A
RAG Pipelines for Agents
  • Retrieval-Augmented Generation: chunking, embedding, indexing

  • Hybrid search: keyword + semantic + reranking
  • Lab: Build a RAG-powered knowledge agent for enterprise docs
  • Agent guardrails: content safety, prompt injection defense
Session B
Capstone: Production Agent System
  • Team project: Design and build a complete agentic AI system
  • Architecture review: scalability, monitoring, cost optimization
  • CI/CD for agents: testing, versioning, deployment pipelines
  • Team presentations and expert feedback
  • Certification paths and next steps
