Deep Learning & Computer Vision

Course Overview

This 5-day advanced bootcamp covers deep learning theory and practice — from neural network fundamentals through CNNs, RNNs, and Transformer architectures to production computer vision systems. Participants implement 6 end-to-end projects using TensorFlow, PyTorch, and Hugging Face across image classification, object detection, and generative vision tasks.

From math to production. This bootcamp bridges the gap between understanding neural network theory and deploying real vision systems. You'll implement architectures from scratch, fine-tune pretrained models, and deploy inference pipelines — covering the full stack that enterprise AI teams need for banking fraud detection, manufacturing quality control, healthcare imaging, and document intelligence.

Learning Outcomes

1
Build neural networks from scratch — implement perceptrons, activation functions, backpropagation, and optimization using NumPy and then TensorFlow/PyTorch
2
Master CNNs for computer vision — implement and fine-tune architectures like ResNet, VGG, EfficientNet for classification, detection, and segmentation
3
Apply sequence models — build RNNs, LSTMs, and GRUs for time-series forecasting, text generation, and sequential data processing
4
Understand Transformer architectures — attention mechanisms, BERT, Vision Transformers (ViT), and their applications in NLP and vision
5
Build production vision pipelines — object detection (YOLO), image segmentation, OCR, and document intelligence using OpenCV and Azure AI Vision
6
Deploy deep learning models — ONNX export, TensorRT optimization, containerized inference with FastAPI, and cloud deployment

Prerequisites

Python programming (intermediate), basic machine learning knowledge (Bootcamp 4 or equivalent), linear algebra fundamentals (matrices, vectors). GPU-enabled environment provided during labs.

Course Curriculum

5-Day Curriculum

Interactive format with live coding, hands-on labs, and project-based learning. Schedule and session order are fully customizable based on your organization's needs and time zones.

01
Neural Network Foundations — From Math to Code
Lecture + Live Coding + Hands-On Labs
Session A
Deep Learning Fundamentals
  • Neurons, activation functions (ReLU, sigmoid, softmax)
  • Forward propagation and loss functions
  • Backpropagation and gradient descent (SGD, Adam, RMSprop)
  • Lab: Build a neural network from scratch using NumPy
  • Regularization: dropout, batch normalization, L1/L2
Session B
TensorFlow & PyTorch Essentials
  • TensorFlow 2.x: tensors, layers, Sequential and Functional APIs
  • PyTorch: tensors, autograd, nn.Module, DataLoaders
  • Training loops, callbacks, checkpointing, TensorBoard
  • Lab: Build and train an MNIST classifier with TensorFlow
  • Lab: Build the same classifier with PyTorch — compare workflows
02
Convolutional Neural Networks — Image Intelligence
Live Coding + Hands-On Labs + Architecture Workshop
Session A
CNN Architecture & Image Classification
  • Convolution, pooling, stride, padding — visual intuition
  • Classic architectures: LeNet, AlexNet, VGG, ResNet
  • Modern architectures: EfficientNet, MobileNet
  • Lab: Build a custom CNN for medical image classification
  • Lab: Transfer learning with ResNet on a custom dataset
Session B
Advanced CNN Techniques
  • Data augmentation strategies for small datasets
  • Fine-tuning pretrained models on domain-specific data
  • Grad-CAM: visualizing what CNNs learn
  • Lab: Fine-tune EfficientNet for manufacturing defect detection
  • Lab: Build an image similarity search engine

03
Sequence Models & Transformers
Lecture + Live Coding + Hands-On Labs
Session A
RNNs, LSTMs & Sequence Processing
  • Recurrent Neural Networks: architecture and limitations
  • LSTMs and GRUs: solving the vanishing gradient problem
  • Bidirectional and stacked RNN architectures
  • Lab: Time-series forecasting for stock price prediction
  • Lab: Text generation using character-level LSTM
Session B
Transformer Architecture & Attention
  • Self-attention mechanism — intuition and mathematics
  • Multi-head attention, positional encoding, encoder-decoder
  • BERT, GPT architecture overview (for understanding, not training)
  • Vision Transformers (ViT): applying Transformers to images
  • Lab: Fine-tune a Hugging Face BERT model for text classification
04
Computer Vision — Detection, Segmentation & OCR
Live Coding + Hands-On Labs + Industry Projects
Session A
Object Detection & Segmentation
  • Object detection architectures: YOLO (v8/v11), SSD, Faster R-CNN
  • Instance segmentation with Mask R-CNN
  • Semantic segmentation with U-Net
  • Lab: Build a real-time object detection system with YOLOv8
  • Lab: Vehicle and pedestrian detection for traffic analysis
Session B
OCR, Document AI & Azure Vision
  • OpenCV fundamentals: image processing, contours, transformations
  • OCR pipelines: Tesseract, EasyOCR, Azure Document Intelligence
  • Azure AI Vision: image analysis, face detection, custom vision
  • Lab: Build a cheque/ID document extraction pipeline
  • Lab: Invoice processing with Azure Document Intelligence
05
Generative Vision, Deployment & Capstone
Capstone Project + Deployment Lab + Presentations
Session A
Generative Models & Advanced Vision
  • Autoencoders: vanilla, variational (VAE), for anomaly detection
  • GANs: generator-discriminator architecture, DCGAN
  • Diffusion models overview (Stable Diffusion, DALL-E concepts)
  • Lab: Build a VAE for anomaly detection in manufacturing
  • Lab: Image captioning with vision-language models
Session B
Model Deployment & Capstone
  • ONNX export and model optimization (TensorRT, quantization)
  • Serving vision models with FastAPI and Docker
  • Edge deployment considerations: mobile, IoT, embedded
  • Capstone: End-to-end vision project — team presentations
  • Certification paths and next steps
Flexible & Customizable: Session order, timing, depth, and industry focus can be tailored to your organization's specific requirements. Content is regularly updated to reflect the latest AI developments.
