AI on Google Cloud

Course Overview

This 5-day hands-on bootcamp covers the full Google Cloud AI and ML platform — from Gemini API and Vertex AI through Document AI, Dialogflow CX, AutoML, and BigQuery ML. Developers build 6 production-ready AI solutions using Google's AI stack, learning to leverage its strengths in multimodal AI, data analytics integration, and open-source model hosting.

Google's AI stack is uniquely powerful for data-rich organizations. Gemini's multimodal capabilities, Vertex AI's integrated ML platform, BigQuery ML's SQL-based modeling, and Dialogflow's conversational AI — this bootcamp covers the entire ecosystem hands-on. You'll understand how Google Cloud AI competes with and complements Azure and AWS, and when GCP is the right choice for your workload.

Learning Outcomes

1
Build with Gemini API — multimodal prompting (text, image, video, audio), function calling, grounding, system instructions, and streaming
2
Master Vertex AI platform — model deployment, Vertex AI Pipelines, model registry, endpoints, evaluation, and experiment tracking
3
Build with AutoML & BigQuery ML — train custom models without code using AutoML, and run ML directly in SQL with BigQuery ML
4
Implement conversational AI — build enterprise chatbots and voice agents with Dialogflow CX, including multi-turn flows and webhook integration
5
Process documents at scale — Google Document AI for invoices, contracts, forms, and custom document types with extraction and classification
6
Deploy and monitor AI — Vertex AI endpoints, model monitoring, A/B testing, Cloud Functions for serverless inference, and cost optimization

Prerequisites

Python programming (intermediate), basic GCP familiarity (console, projects, IAM). SQL knowledge for BigQuery labs. A GCP account with billing enabled is recommended (free tier credits work for labs).
