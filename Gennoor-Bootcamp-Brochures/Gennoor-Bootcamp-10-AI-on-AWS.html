AI on AWS

Course Overview

This 5-day hands-on bootcamp covers the full AWS AI and ML platform — from Amazon Bedrock and SageMaker through Lex, Comprehend, Textract, Rekognition, and serverless AI patterns. Developers build 6 production-ready AI solutions using AWS's AI stack, learning to leverage its strengths in model choice, serverless architecture, and global infrastructure.

AWS offers the widest choice of AI models and the deepest serverless integration. Amazon Bedrock gives you access to Claude, Llama, Mistral, Titan, and more through a single API. SageMaker provides the most mature ML platform for custom training and deployment. This bootcamp covers the entire ecosystem — from managed AI services for rapid development to custom ML pipelines for full control.

Learning Outcomes

1
Build with Amazon Bedrock — access Claude, Llama, Mistral, and Titan through a unified API with guardrails, knowledge bases, and agents
2
Master Amazon SageMaker — custom model training, hyperparameter tuning, model registry, real-time and batch inference endpoints
3
Implement RAG on AWS — Bedrock Knowledge Bases with S3, OpenSearch Serverless vector search, and retrieval-augmented generation
4
Build conversational AI — Amazon Lex bots with Lambda fulfillment, multi-turn flows, and integration with Connect for voice
5
Process documents and media — Amazon Textract for document extraction, Comprehend for NLP, Rekognition for image/video analysis
6
Deploy serverless AI — Lambda + API Gateway for inference, Step Functions for ML workflows, and EventBridge for event-driven AI

Prerequisites

Python programming (intermediate), basic AWS familiarity (console, IAM, S3). Experience with REST APIs. An AWS account with Bedrock model access is recommended (free tier works for most labs).

01
Amazon Bedrock — Generative AI on AWS

Bedrock Fundamentals

  • SageMaker Pipelines: automated ML workflows

  • Lab: Set up model monitoring with automated retraining
