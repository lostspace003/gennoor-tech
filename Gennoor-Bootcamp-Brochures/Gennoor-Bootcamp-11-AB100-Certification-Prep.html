Course Overview

This 5-day intensive bootcamp prepares developers and architects to pass the Microsoft AB-100: Agentic AI Business Solutions Architect certification exam. Every session maps directly to official exam domains. Participants gain deep understanding of designing, implementing, and governing agentic AI solutions using Microsoft Copilot Studio, Azure AI, Semantic Kernel, and Power Platform — with hands-on labs and practice assessments throughout.

Taught by one of the first AB-100 certified professionals globally. This is not a theoretical course — you'll learn from an instructor who has passed the exam, built production agentic AI systems, and delivered enterprise agent deployments. Every topic, lab, and practice question is aligned to the official Microsoft skills measured outline for AB-100.

5-Day Curriculum

Every session maps to official AB-100 exam domains. Includes hands-on labs, case studies, and practice assessments. Schedule is fully customizable based on your organization's needs.

01
Exam Domain: Design Agentic AI Solutions
Lecture + Case Studies + Architecture Workshop
Session A
Agentic AI Fundamentals & Architecture
  • What makes an AI system "agentic" — reasoning, planning, tool use
  • Agent architecture patterns: single-agent, multi-agent, orchestrator
  • Microsoft agentic AI stack: Copilot Studio, Semantic Kernel, Azure AI Agent Service
  • Use case identification: when to use agents vs chatbots vs automation
  • Practice: Analyze 5 business scenarios and design agent solutions
Session B
Solution Design & Requirements Mapping
  • Translating business requirements into agent architecture decisions
  • Selecting the right platform: Copilot Studio vs custom Semantic Kernel vs hybrid
  • Multi-agent collaboration patterns: delegation, specialization, consensus
  • Human-in-the-loop design: approval gates, escalation, feedback loops
  • Practice: Design a complete agentic solution for a BFSI use case
02
Exam Domain: Implement with Copilot Studio
Hands-On Labs + Implementation Workshop
Session A
Copilot Studio Agent Development
  • Copilot Studio architecture: topics, entities, variables, plugins
  • Generative AI in Copilot Studio: knowledge sources, grounding
  • Custom plugins: Power Automate connectors, HTTP actions
  • Lab: Build a generative answers agent with knowledge grounding
  • Lab: Build a multi-topic agent with conditional routing
Session B
Advanced Copilot Studio Patterns
  • Copilot Studio plugins: custom connectors, OpenAPI specs
  • Authentication: SSO, OAuth, service principal patterns
  • Adaptive Cards for rich agent responses
  • Lab: Build an agent with Power Automate approval workflows
  • Lab: Build a Copilot agent with custom API plugin integration

Live Coding + Hands-On Labs
