01
AI Agent Foundations — LangChain & Tool Calling
Hands-On Labs + Workshop
Session A
AI Agent Foundations
  • Agent architectures: ReAct, Plan-and-Execute, Reflexion

LangChain & Tool Calling
  • Custom tools: web search, database queries, API calls

LangGraph — Stateful Multi-Step Workflows
Demos + Labs + Discussion
Session A
LangGraph
  • Graph-based orchestration: nodes, edges, state management
  • Conditional routing, branching, and persistence (checkpoints)
  • Human-in-the-loop: approval gates, feedback loops
Session B
Stateful Multi-Step Workflows
  • Error handling, retries, and streaming responses
  • Lab: Loan approval agent with human review step
