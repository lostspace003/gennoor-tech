Purpose & Objective

Why it exists

Religious education for children is usually passive — text to read, videos to watch. Kids disengage. Safar-e-Anbiya makes learning the stories of the prophets active, beautiful and rewarding — something a child chooses to open, the way they open a game.

🎯

The objective

Turn the lives of all 25 prophets into an interactive adventure a 5–10 year old wants to return to every day.

🕹️

The approach

Each prophet is a level: story → moral choice → consequence → the real Qur'anic ayah → quiz → reward.

👨‍👩‍👧

Who it's for

Muslim families & children. Parent-managed accounts, child profiles, and a generated avatar each.

Inside the experience

Cinematic by design

  • 🌌25 distinct night skies. Each prophet has its own palette, aurora wash and moon placement — every stage feels different.
  • 🎞️Moving illustrations. Continuous Ken Burns drift + per-scene cross-fades make every story beat feel alive.
  • 🗣️Bilingual neural narration. English & Urdu (male/female) Azure voices with live word-by-word highlighting — English uses familiar Biblical prophet names (Abraham, Moses…).
  • 🕋The real ayah. 3–4 authentic Qur'anic verses per prophet — qari recitation plus spoken meaning.
  • 🏆Noor gamification. Level ladder, 1–3 stars, badge gallery, 🔥 daily streak, confetti & chime SFX.
  • 👨‍👩‍👧Parent accounts & safe leaderboard. Email-OTP sign-in, child profiles, and a privacy-first board (first name + age only — no photo, surname or country).

A look at the code

Readable scene styling

The whole UI is styled with inline CSS strings parsed by a tiny s() helper — it turns "border-radius:30px;color:#f4eede" into a React style object, keeping hundreds of one-off scene styles legible inline.

app/components/ProphetsJourney.jsx
function s(str) {
  const o = {};
  if (!str) return o;
  str.split(";").forEach((decl) => {
    const i = decl.indexOf(":");
    if (i < 0) return;
    const prop = decl.slice(0, i).trim();
    const val  = decl.slice(i + 1).trim();
    if (!prop || !val) return;
    // border-radius → borderRadius
    const key = prop.replace(/-([a-z])/g, (_, c) => c.toUpperCase());
    o[key] = val;
  });
  return o;
}
